# Личный план на апрель (с заменой батарей)
---

## Неделя 1 (1–6 апреля)

### 1 апреля, вторник — Смена 1 (дневная)
- **09:00–21:00** — работа
- **21:30–22:00** — поиск компаний по окнам (0,5 ч)
- **22:00–22:30** — сохранить референсы по ремонту (0,5 ч)

---

### 2 апреля, среда — Смена 2 (ночная)
- **21:00–09:00** — работа (ночная смена)
	- Умыться, почистить зубы
	- Обшить стулья
- **14:00–16:00** — программирование (2 ч)

---

### 3 апреля, четверг — Выходной
- **10:00–12:00** — программирование (2 ч)
- **12:00–13:00** — обед
- **13:00–14:00** — поиск идей ремонта: систематизация референсов (1 ч)
- **15:00–17:00** — программирование (2 ч)
- **17:00–18:00** — звонки в компании по окнам, согласование замера (1 ч)

---

### 4 апреля, пятница — Выходной (до вечера)
- **10:00–12:00** — программирование (2 ч)
- **12:00–13:00** — обед
- **13:00–14:00** — Забрать справки с переливания крови, поиск идей ремонта: варианты отделки (0,5 ч)
- **14:00–15:00** — термопаста: поиск инструкции, просмотр видео, подготовка (1 ч)
- **15:30–17:00** — уборка генеральная (1,5 ч)
- **17:00–18:30** — свободное время
- **с 19:00** — личное время с девушкой

---

### 5 апреля, суббота — Смена 1 (дневная)
- **09:00–21:00** — работа
- **18:00–20:00** — приготовление нового блюда (вместе с девушкой)  
  *Паста с томатно-сливочным соусом и шпинатом*

---

### 6 апреля, воскресенье — Смена 2 (ночная)
- **21:00–09:00** — работа (ночная смена)
- **весь день** — личное время (сон, восстановление)

---

### 9 апреля, среда — Смена 1 (дневная)

- **10:00–12:00** — программирование (2 ч)

- **14:30–15:30** — составить список вещей в отпуск
- **15:30–16:30** — батареи: уточнить дату установки (записать в календарь) (0,5 ч)

### 20 апреля, воскресенье — Выходной

- **14:00–15:00** — финальная проверка документов, зарядных устройств
- **15:00–16:00** — вынести мусор, проверить холодильник, закрыть окна
- **остаток дня** — отдых перед отпуском

| Задача                                                    | Дедлайн  | ✅   |
| --------------------------------------------------------- | -------- | --- |
| Определиться с типом радиаторов (биметалл/алюминий/чугун) | 8–11 апр | ☐   |
| Рассчитать мощность на каждую комнату (по площади)        | 11 апр   | ☐   |
| Найти компанию для замены                                 | 8–11 апр | ☐   |
| Вызвать замерщика                                         | 16 апр   | ☐   |
| Получить расчет (радиаторы + трубы + работа)              | 16 апр   | ☐   |
| Подписать договор, предоплата                             | 17 апр   | ☐   |
| Уточнить дату установки (после окон)                      | 18 апр   | ☐   |
